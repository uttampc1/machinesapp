📋 Machine Management — Command Line Tools
Reserve / Release a Machine
SSH into the machine and run:
machine-reserve to reserve  │  machine-release to release  │  machine-status to check status
These commands update the machine's MOTD, broadcast to logged-in users via wall, and update this inventory database.
The machine itself is the source of truth for reservation status.
🔧 Setup — If scripts are not installed on your machine
1. Clone the repo:  git clone https://github.com/uttampc1/machine-status-scripts.git
2. Follow the setup instructions in the repo's README.md
